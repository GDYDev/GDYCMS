Detailed Description

This controller create site interface.

View:Index.cshtml (empty view that receive model (ViewerModel class (GDYCMS.Models.ViewerModel) ) and use layout:_BaseLayout.cshtml, which also use same model. _BaseLayout.cshtml use partial views and take them model. Partial views are:

  • _CentralContentPartial.cshtml Main content
  • _LeftMenuDownPartial.cshtml Left down menu
  • _LeftMenuUpPartial.cshtml Left up menu
  • _RightMenuDownPartial.cshtml Right menu down
  • _RightMenuUpPartial.cshtml Right up menu
  • _UpMenuPartial.cshtml Up menu
